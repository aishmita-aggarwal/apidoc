Sinatra API Docs

Allows you to add inline documentation to a Sinatra app and have a web-based documentation browser be available to you.

Documentation Format

register Sinatra::APIDocs

section  "Applications"

desc     "Create an application."
param    "name",  "the name of the application to create"
param    "stack", "[OPTIONAL] the stack on which to create the application"

request  "POST /apps.json\"\nname=example&stack=bamboo-ree-1.8.7\n"

response "{\n\"id\":         1,\n\"name\":       \"example\",\n\"owner\":      \"user@example.org\",\n\"created_at\": \"Sat Jan 01 00:00:00 UTC 2000\",\n\"stack\":      \"bamboo-ree-1.8.7\",\n\"slug_size\":  1000000,\n\"repo_size\":  500000,\n\"dynos\":      1,\n\"workers\":    0\n}\n"

post "/apps.json" do
  ...
end

Web-based browser

map "/docs" do
  run Sinatra::APIDocs::Viewer.new(YourSinatraApp)
end
